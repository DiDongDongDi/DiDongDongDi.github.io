eclipse Unhandled event loop exception No more handles 错误解决

Posted on 2019-09-03 In Problem Views:
Symbols count in article: 103 Reading time ≈ 1 mins.

将tomcat安装到非root用户下后, 同时将eclipse也安装到同一用户下, 使用该用户的权限启动后, 不再出现该问题

建议以后统一将自定义安装软件安装到普通用户目录下, 而不是root用户
